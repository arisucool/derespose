Derespose について

Derespose へようこそ

Derespose (でれすぽーず) は、デレスポのポーズを探すための非公式サービスです。
デレスポでAR写真を撮影されている P の皆さまに役立てていただけることを願って生まれました。

ご意見などは、 @arisucool までお気軽にお寄せください。

ご不明な点がある場合は、 FAQ もご確認ください。
また、現在対応しているポーズ (楽曲) については、ポーズの対応状況 をご覧ください。

利用規約・プライバシーポリシー

1. 本サービスおよび第三者の関係性について

デレスポ (正式名称: アイドルマスター シンデレラガールズ スターライトスポット) は、株式会社バンダイナムコエンターテインメント様、株式会社Cygames様の製品および商標です。

一方、Derespose (以下、本サービス) は、同社と一切関係のない個人により趣味として開発しております。
同社に問い合わせをするなど、迷惑をかける行為は決して行わないよう、お願いいたします。

2. 免責事項

本サービスの使用によって、いかなるトラブルが発生した場合も、開発者は一切の責任を負いかねます。
※ 本サービスはデレスポと連携するものではなく、デレスポの動作に一切の影響を与えることはありません。

本サービスは、予告なく休止または終了する場合があります。ご了承ください。

3. ポーズ認識検索で扱う情報について

本サービスのポーズ認識検索では、お使いの端末のカメラを使用して、あなたのポーズを認識します。
カメラの映像は、お使いの端末内で処理されるため、サーバへの送信や記録等は行いません。
ただし、認識したポーズ情報 (映像・画像・個人情報等を含まないもの) は、サーバに送信し、一致するタグの取得や精度改善のために使用します。

4. タグ検索で扱う情報について

本サービスのタグ検索では、選択されたタグの情報 (例: "ダブルピース") をサーバへ送信することで、検索を行います。

5. タグ編集で扱う情報について

本サービスでは、他のユーザの役に立つよう、検索したポーズにタグを付けることができます。
編集されたタグの情報は、サーバへ送信および記録し、全てのユーザと共有します。

6. アクセスログの取得について

本サービスでは、他のいかなるサービスと同様に、サーバ側でアクセスログを取得しており、IPアドレスを記録しています。
このアクセスログは、原則、12ヶ月経過後に自動的に削除します。

7. 情報の監視および削除について

本サービスにて保管しているいかなる情報も、管理者による確認および判断によって、予告や作成者の了承なしに削除することができるものとします。
特に、長期間にわたって利用実績のない情報や、健全なサービス運用にそぐわない不適切な内容と判断した情報は、積極的に削除または非公開への変更を行います。
持続可能かつ健全なサービス運用のために、皆様のご理解ご協力をお願いいたします。

不適切な内容について、以下に一例を挙げます。
例えば、タグやマイリストの名称などにおいて、これらを使用した場合、ただちに措置を行います。

  • アイドルまたは関係各所に対する誹謗中傷、またはイメージを毀損する内容
  • センシティブな内容
  • 悪戯目的の内容
  • その他、管理者が不適切と判断した内容

また、本サービスにて登録されたユーザ情報およびポーズリストは、ユーザ自身の申請によって、いつでも削除することが可能です。

8. Twitter アカウントとの連携 (任意) について

本サービスでは、いかなるユーザ登録またはログインをせずとも、基本機能を利用いただけます。
しかし、Twitter アカウントにてログインすることで、追加機能をご利用いただけます。

本サービスにて、Twitter アカウントを用いてログインした際、Twitter 社から当該アカウントの情報提供が行われます。
さらに本サービスのデータベースでは、提供されたアカウント情報をもとに、ユーザ情報の登録または更新を行います。
尚、 Twitter API にて使用する権限は、読み取り権限のみであり、Twitter に対する書き込みは行いません。

Twitter API より取得した情報のうち、使用・保管する項目は、以下の通りです。
これらの項目は、本サービスの "ポーズリスト" (マイリスト) 機能など、追加機能にて利用します。

  • ユーザID (id)  -  (Twitter ユーザを識別する一意の数字)
  • ユーザ名 (username)  -  (例: "arisucool")
  • 表示名 (displayName)  -  (例: "arisu.cool 🍓")
  • アクセストークン (accessToken)  -  (Twitter API から提供される情報ですが、現状は使用しておりません。)

本サービスにて登録されたユーザ情報は、作成者自身の申請により、いつでも削除することが可能です。

9. ポーズリスト機能 (マイリスト) で扱う情報について

本サービスでは、ポーズリストを作成し、検索したポーズを登録できます。
作成したポーズリストは、選択した公開範囲が "公開" の場合、全てのユーザに対して自動的に共有(公開)します。
また、"限定公開" の場合、自動的に公開することはありませんが、 作成者は当該ポーズリストのURLを用いて、第三者に対してポーズリストを共有できます。
いずれの場合も、共有されたポーズリストには、作成者の Twitter アカウントのユーザ名を表示します。

本サービスにて作成されたポーズリストは、作成者自身の操作により、いつでも削除することが可能です。

10. プライバシーポリシーの改定について

これらのポリシーは、今後の機能追加などに伴って変更される可能性があります。あらかじめご了承ください。
